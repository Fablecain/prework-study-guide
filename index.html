HTML

  • The head element contains info about the page.
  • The body element represents the visible content shown to the user.
  • The H1 element is the main header of the webpage and can only be used once per page.
  • We can add as many "li" elements as we want to our "ul".
  • "ul" creates an unordered list of items

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The "href" attribute specifies the location URL of the external resource.
  • The "rel" attribute specifies the relationship between the current document and the linked document resource.

Git

  • git status is a way we check what branch we are currently on
  • git checkout -b "branch-name" creates a new branch and switches to it
  • git checkout main switches us to our main branch
  • git pull origin main pulls the latest branch

JavaScript

  • Console.log is used to output a message to the web console.
  • Console.log outputs the message by adding in an argument.
  • To declare a variable we use the "Var" keyword in JS.
  • After inputting "Var", the word following will be our variable.
  • The variable contains a string that is contained within quotation marks.
  • An Array is a single variable that is used to hold a group of data that is typically related in some way.
